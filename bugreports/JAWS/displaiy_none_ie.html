IE/JAWS Issues related to Hidden Links

When display:none or aria-hidden='true' is directly placed on a link, IE will expose the link as having the 'invisible' state but JAWS will still announce the link. If these attributes are placed on a parent of the link, the parent and the link are correctly ignored by JAWS. Note that virtual navigation has to be used to detect the issues for links 2 and 3. The issue extends to link lists as well. Adding visibility:hidden has no effect, as shown below.
