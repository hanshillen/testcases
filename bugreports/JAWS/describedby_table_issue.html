JAWS Issue: aria-describedby Double Announcement

Issue Description

Some versions of JAWS announce aria-describedby content twice when an element receives focus.

Use case:

The text field below is described by the text below it using aria-describebdy

Your name must be two characters or more.

Test Results:

JAWS 11:
IE: JAWS reads description text once
FF: JAWS reads description text once 
JAWS 12: 
IE: JAWS reads description text once
FF: JAWS reads description text once 
JAWS 13: 
IE: JAWS reads description text twice
FF: JAWS reads description text twice
JAWS 14:
IE: JAWS reads description text twice
FF: JAWS reads error description once
