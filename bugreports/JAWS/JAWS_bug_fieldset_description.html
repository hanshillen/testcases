JAWS test case: Aria-describedby text on fieldset not read in IE

The fieldset below has a legend element as well as an additional description provided by aria-describedby. When moving focus into the fieldset, the expectation is that both the legend text and descriptive text will be announced. Instead, the text associated with aria-describedby is ignored in Internet Explorer. In Firefox the description is correctly announced by JAWS after the legend text.
