HFP Notes

Known Accessibility Issues

VoiceOver does not announce the dialog when focus is moved into a field inside of it.

Notes for Developers

No accessibility related notes.

Notes for Testers

When the dialog is created, ensure that:

  • Focus is moved to a field inside the dialog
  • The screen reader announces the dialog before announcing the focused element
  • It is not possible to tab out of the dialog
  • It is possible to close the dialog using Escape or a "close" button
