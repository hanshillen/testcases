    Please note that uploading a second version of the template will replace the previously uploaded version.

    Please use the following guidance to complete the Essential Community Providers template:

    • The template provides a list of ECPs that have been identified by CMS as meeting the definition of an ECP.
    • The template will allow the applicant to identify each provider network it intends to utilize for its QHPs and to select each ECP with which it has an executed contract in each network.
    • The applicant may also write in additional ECPs if these ECPs do not appear on the list and meet the definition of an ECP as set forth in 155 CFR 156.235(c).
    • The applicant should identify any network that is different for its individual and small group market as a separate network.
