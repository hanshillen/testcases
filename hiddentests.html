Test cases for hiding content in an accessible way

Each sample contains the following:

  1. A link with some hidden text in the middle
  2. Hidden static content
  3. A Hidden focusable button
  4. A text field that has a associated hidden label

Sample 1 

	.sample1 .hidden-accessible {
		position: absolute !important;
		clip: rect(1px 1px 1px 1px); /* IE6, IE7 */
		clip: rect(1px, 1px, 1px, 1px);
	}
This is a partially hidden link
