Example 2: Workaround Fix

The problem is that JAWS 12 incorrectly announces the main containers accessible name before announcing the text field. Since the main landmark is not labeled, JAWS uses the container's text contents as a fallback accessible name. To fix, add title='' to the role='main' element. This will set the main landmark's accessible name to an empty string. JAWS will stil announce it but because it's empty it won't actually say anything.
